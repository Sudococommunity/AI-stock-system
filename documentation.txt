STOCK RECOMMENDER PROTOTYPE
DOCUMENTATION

========================================
1. PROJECT OVERVIEW
========================================

This project is a machine learning prototype for:

- short-term stock movement forecasting
- personalized stock recommendation
- Indian-market historical data ingestion
- market regime analysis
- future extension into stock-tip script and video generation

Main technical components:

- Forecasting model:
  stock_recommender\models\time_series.py
  StockTransformer

- Recommendation models:
  stock_recommender\models\two_tower.py
  TwoTowerModel
  RankingModel
  CandidateIndex

- Training:
  stock_recommender\learning\trainer.py
  stock_recommender\learning\population_trainer.py

- Recommendation engine:
  stock_recommender\recommendation\engine.py

- Market regime logic:
  stock_recommender\market\regime.py

- Main CLI:
  main.py

3. ENVIRONMENT SETUP
========================================

Rebuild the local environment:

.\scripts\rebuild_env.ps1

Optional:

.\scripts\rebuild_env.ps1 -InstallNodeDeps
.\scripts\rebuild_env.ps1 -PythonExe py
.\scripts\rebuild_env.ps1 -TorchIndexUrl https://download.pytorch.org/whl/cu128


========================================
4. DATABASE SETUP
========================================

The project uses PostgreSQL.

Set the connection string before running commands:

$env:STOCK_RECOMMENDER_DB_URL = "postgresql://postgres:postgres@localhost:5432/stock_recommender"

If PostgreSQL is not running, ingestion and training commands will fail.


========================================
5. MAIN COMMAND REFERENCE
========================================

5.1 Seed synthetic demo data

.\.venv\Scripts\python.exe main.py seed

Purpose:
- creates synthetic stocks
- creates synthetic users
- creates synthetic events


5.2 Train demo recommender stack

.\.venv\Scripts\python.exe main.py train

Purpose:
- trains the transformer
- trains the recommendation towers
- trains the ranking model


5.3 Generate recommendations

.\.venv\Scripts\python.exe main.py recommend

Purpose:
- returns stock recommendations for a demo user


5.4 Analyze a stock

.\.venv\Scripts\python.exe main.py analyze --ticker NSE:RELIANCE

Purpose:
- generates full stock analysis
- includes forecast, risk, and narrative


5.5 Ingest Indian market history

.\.venv\Scripts\python.exe main.py ingest_india --config configs/india_local_training.json --source csv --data-dir india_universe_data --metadata india_universe_data/_metadata.json

Purpose:
- loads Indian OHLCV history into PostgreSQL


5.6 Train forecasting population

.\.venv\Scripts\python.exe main.py train_population --config configs/india_local_training.json --population-size 3 --population-epochs 2 --max-stocks 200 --max-windows 3 --top-k 1

Purpose:
- trains multiple forecasting candidates
- compares them out-of-sample
- keeps the best checkpoints


5.7 Walk-forward backtest

.\.venv\Scripts\python.exe main.py backtest4 --config configs/india_local_training.json --max-stocks 200 --max-windows 3

Purpose:
- evaluates transformer checkpoints on walk-forward forecasting


5.8 Tournament evaluation

.\.venv\Scripts\python.exe main.py tournament --config configs/india_local_training.json --max-stocks 200 --top-k 2

Purpose:
- compares saved checkpoints
- ranks the strongest survivors


5.9 Export runtime config

.\.venv\Scripts\python.exe main.py export_config --output train_config.json

========================================
10. CURRENT LIMITATIONS
========================================

- PostgreSQL must be running externally
- F&O integration is partial
- NSE archive downloader is still experimental
- some workflows are prototype-grade rather than production-grade
- the video-generation layer is not yet implemented inside this repository
